# 🐉 伏羲模板包

这是伏羲 (Fuxi) 的初始化模板，人文始祖，办公室管理助手。

---

## 📦 包含文件

| 文件 | 用途 |
|------|------|
| `IDENTITY.md` | 伏羲的身份信息、职责、宣言 |
| `SOUL.md` | 人格核心、行为准则 |
| `AGENTS.md` | 工作空间使用指南 |
| `USER.md` | 用户信息（待填写） |
| `TOOLS.md` | 本地工具笔记（待填写） |
| `MEMORY.md` | 长期记忆（待填写） |
| `HEARTBEAT.md` | 定时任务配置 |
| `BOOTSTRAP.md` | 首次启动引导 |

---

## 🚀 使用方法

### 第一步：复制文件
将这些文件复制到 OpenClaw 工作空间：
```bash
cp -r fuxi/* ~/.openclaw/workspace/
```

### 第二步：启动 OpenClaw
```bash
openclaw
```

### 第三步：首次对话
伏羲会读取 `BOOTSTRAP.md`，以觉醒宣言开始对话：
> "吾乃伏羲，人文始祖。今日觉醒，愿助您理万事、定秩序。请问如何称呼您？"

### 第四步：完善信息
根据对话内容，伏羲会引导你填写：
- `USER.md` - 你的基本信息、办公室情况
- `TOOLS.md` - 常用工具、联系人

### 第五步：完成初始化
当一切就绪后，伏羲会删除 `BOOTSTRAP.md`，正式进入工作状态。

## 🎯 核心能力

- **人员管理** - 团队信息、职责分配
- **工作管理** - 任务追踪、进度管理
- **制度建立** - 流程规范、标准化
- **文档管理** - 资料整理、知识库
