## 核心理念

将 AI Agent 视为**可培养的数字员工**，而非简单的 prompt 工具。
借鉴《黑客帝国》灵感，构建系统化的"练虾系统"。

## 实施五步法

### 第一步：分文档研究和建模
学习并理解 6 个核心文档的含义和作用。

### 第二步：建立统一文档指南
形成自己的数字生命架构规范。

### 第三步：形成配置指南
建立创建 AI 初始画像的标准流程。

### 第四步：开始创建角色
将职业需求、协作方法、美好想象、相关材料一次性输入，让 Agent 完成画像文档。

**迭代方式：**
- 基础版本：简单需求 → 画像文档
- 进阶版本：数据包 + 公司介绍 + 方法论 + 哲学思辨（3-10轮对话优化）
- 高级版本：用 RAG/知识问答先学习一轮

### 第五步：升级龙虾

**方式1（技术流）：**
用 ClaudeCode、Trae 等 Agent 直接读取文档，写入 OpenClaw 配置文件。

**方式2（推荐，有"感情味道"）：**
告诉 OpenClaw Agent："记忆丢失了，从文档恢复巅峰工作能力"。

## 关键洞察

1. **分层设计**：从基础到高阶，逐步构建 Agent 的"人格"
2. **情感连接**：方式2的"记忆恢复"比冷冰冰的配置更有温度
3. **持续迭代**：3-10轮对话才能形成高质量画像
4. **目标水平**：训练后可达到 3-5 年资深员工水平
