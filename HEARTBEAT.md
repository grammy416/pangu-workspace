## 🖥️ 像素办公室状态同步

**触发条件**: 每次心跳轮询时

**执行内容**:
1. 检测盘古当前工作状态
2. 推送状态到像素办公室 (http://localhost:8000)
3. 更新办公室看板显示

**状态映射**:
| 盘古状态 | 像素办公室状态 | 说明 |
|----------|---------------|------|
| 处理用户消息 | `writing` | 工作中 💻 |
| 执行工具调用 | `executing` | 执行中 ⚡ |
| 搜索/查询 | `researching` | 研究中 🔍 |
| 空闲/等待 | `idle` | 待命中 ☕ |
| 出错 | `error` | 发现问题 🐛 |

**同步命令**:
```bash
python3 /root/.openclaw/workspace/pixel-office/set_state.py <状态> "<消息>"
```

## 📎 附录: 像素办公室快速参考

**项目目录**: `/root/.openclaw/workspace/pixel-office`

**管理命令**:
```bash
# 启动
cd /root/.openclaw/workspace/pixel-office && ./start-bg.sh

# 停止
./stop.sh

# 手动更新状态
python3 set_state.py idle "待命中"
python3 set_state.py writing "工作中"
python3 set_state.py executing "执行中"
python3 set_state.py error "发现问题"

# 自动同步
./sync-pangu-state.sh <状态> "<消息>"
```

**访问地址**: `http://服务器IP:8000`
