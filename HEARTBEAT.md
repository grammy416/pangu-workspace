## 📦 01:00 每日备份任务 (cron触发)

**触发条件**: `backup-daily` 系统事件

**执行内容**:
1. 执行 `/root/.openclaw/workspace/daily-backup-with-report.sh`
2. 检查工作空间变更
3. 提交并推送到 GitHub
4. 发送备份汇报到飞书

---

## ⏰ 23:00 每日简报任务

**触发条件**: 当收到心跳轮询且当前时间接近 23:00 (22:30-23:30)

**执行内容**:
1. 检查今日是否已发送过简报
2. 如未发送，生成并发送当日智能体简报
3. 记录发送时间到日志

**简报内容**:
- Agent 状态总览
- 今日活动统计
- 定时任务状态
- 系统健康度
- 待处理事项

---

## 💰 22:00 投资市场简报任务

**触发条件**: 当收到心跳轮询且当前时间接近 22:00 (21:30-22:30)

**执行内容**:
1. 搜索当日 A 股大盘概况
2. 分析美股收盘情况（如相关）
3. 检查 WATCHLIST.md 中关注股票动态
4. 生成并发送市场简报到飞书

**简报内容**:
- 大盘指数表现（上证/深证/创业板）
- 热点板块分析
- 关注股票动态
- 明日交易提示
